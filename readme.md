# 🕊️ 鸽鸽表情包图鉴 KunMemeHub

这是一个用于收录、整理、分享「鸽鸽」表情包的开源项目。作为互联网文化的重要组成部分，这些表情包见证了社交网络上的趣味创作与梗文化的发展。

📌 本项目仅供娱乐、研究与交流使用，**禁止商业用途**。

---

## 📂 项目结构说明

```bash
KunMemeHub/
├── static/                      # 静态资源（项目logo、Banner等）
├── memes/                      # 表情包主目录
│   ├── 原始图/               # 原始图/截图
│   ├── gif/                    # 动图表情包
│   └── 文字图/             # 带文字的表情包
├── CONTRIBUTING.md             # 贡献指南
├── LICENSE                     # 项目许可协议
└── README.md                   # 项目说明文件
